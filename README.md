# My Portfolio Website

Personal portfolio website showcasing my skills, projects, interests, and passion as a developer.

## About Me

Hi, I'm Harish – a passionate web developer from Coimbatore, India.

## Features

- Responsive design (works great on mobile, tablet, and desktop)
- Modern animations (AOS library)
- Sections include:
  - Introduction
  - Skills & Technologies
  - Projects Showcase
  - Favorites: Football & Fashion inspired cards/quotes
  - Fun personal quote section
  - Contact

## Technologies Used

- HTML5
- CSS3 (with modern layouts – Flexbox/Grid)
- JavaScript
- Font Awesome icons
- AOS (Animate On Scroll)
- Google Fonts
